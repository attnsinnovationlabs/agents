# 14. OOP (Object Oriented Programming)

Four Pillars:

1. Encapsulation
2. Abstraction
3. Inheritance
4. Polymorphism

---

# Class and Object

```python
class Employee:

    def __init__(self,name,salary):
        self.name = name
        self.salary = salary

    def display(self):
        print(self.name,self.salary)

emp = Employee("Tilak",50000)

emp.display()
```

---

# Encapsulation

Hide Data

```python
class Bank:

    def __init__(self):
        self.__balance = 1000

    def get_balance(self):
        return self.__balance

obj = Bank()

print(obj.get_balance())
```

---

## Real-Time Example

Bank Account

Balance cannot be accessed directly.

# Inheritance

Reuse parent class.

```python
class Person:

    def display(self):
        print("Person")

class Employee(Person):
    pass

emp = Employee()

emp.display()
```

# Polymorphism

Same method different behavior.

```python

    def sound(self):
        print("Bark")

        print("Meow")
```

# Abstraction

Hide implementation.

```python
from abc import ABC, abstractmethod

class Vehicle(ABC):

    @abstractmethod
    def start(self):
        pass

class Car(Vehicle):

    def start(self):
        print("Car Started")
```
