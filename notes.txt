Learning Agentic AI is valuable because it enables you to build systems that don't just answer questions—they can **plan, reason, use tools, make decisions, and complete tasks autonomously**.

## What is Agentic AI?

Traditional AI:

```text
User → AI → Response
```

Agentic AI:

```text
User
  |
  v
AI Agent
  |
  +--> Think
  +--> Plan
  +--> Use Tools
  +--> Execute Tasks
  +--> Verify Results
  |
  v
Final Outcome
```

Instead of only generating text, an AI agent can perform actions to achieve a goal.

## Technologies You Will Learn

### OpenAI Agents SDK

Official framework from [OpenAI](https://openai.com/?utm_source=chatgpt.com) for building AI agents.

Features:

* Agents
* Tools
* Handoffs
* Memory
* Multi-agent systems

---

### CrewAI

[CrewAI](https://www.crewai.com/?utm_source=chatgpt.com) allows multiple specialized agents to work together.

Example:

```text
Manager Agent
      |
-------------------
|        |        |
Developer QA  Tester
Agent     Agent
```

---

### LangGraph

[LangGraph](https://www.langchain.com/langgraph?utm_source=chatgpt.com) is useful for building complex workflows and stateful agent systems.

Example:

```text
Start
  |
Research Agent
  |
Decision Agent
  |
Execution Agent
  |
End
```

---

### LangChain

[LangChain](https://www.langchain.com/?utm_source=chatgpt.com) helps connect LLMs to:

* Databases
* APIs
* Documents
* Vector databases
* Tools

---

### AutoGen

[AutoGen](https://microsoft.github.io/autogen/?utm_source=chatgpt.com) from [Microsoft](https://www.microsoft.com/?utm_source=chatgpt.com) supports conversations among multiple agents.

---

### MCP (Model Context Protocol)

[Model Context Protocol](https://modelcontextprotocol.io/?utm_source=chatgpt.com) provides a standard way for AI agents to access external tools, databases, and services.
